\begin{resumo}
	
Este trabalho consiste na construção de um modelo de apoio ao ensino com o auxílio da Consciência Situacional (Situation Awareness - SAW), a SAW é amplamente utilizada na correta compreensão do ambiente e da situação sendo considerada principal precursora do processo de Tomada de Decisão, portanto o seu poder de compreensão do universo cria condições de aplicabilidade e adaptação em Ambientes Virtuais de Aprendizagem (AVA's). Ainda aborda-se a aplicabilidade das técnicas de Mineração de Dados Educacionais, o qual preocupa-se com o desenvolvimento de métodos para tipos únicos de dados que são gerados a partir de ambiente educacionais. Os AVA's por serem extremamente dinâmicos requerem assim ferramentas que se adaptem as variações da interação com o usuário, a EDM age como um motor que processa as informações, entretanto a aplicabilidade muitas vezes restringe-se a situação para qual a técnica foi desenvolvida, a correta compreensão sobre as situações que desenvolvem-se em uma interação do usuário com o AVA pode criar sistemas mais eficientes que atendam com maior assertividade o processo de aprendizagem educacional. A visão consciente sobre o ambiente permite identificar para cada configuração dos dados qual melhor ação a ser tomada. Este modelo deve ser capaz de compreender as informações contextuais disponíveis trabalhando continuamente para criar e atualizar as situações nos quais os métodos da EDM melhor se aplicam e a partir do retorno da execução destes métodos quais prováveis melhores ações devem ser disparadas.


\noindent \textbf{Palavras-chave:} Consciência Situacional, Mineração de Dados Educacionais, Ambientes Virtuais de Aprendizagem, Tomada de Decisão.
\end{resumo}
